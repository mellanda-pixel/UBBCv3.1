# Ultimate Bodies By Carlos - Website

A modern fitness gym website built with Next.js 16, React 19, and Tailwind CSS 4.

## Deployment to Vercel

1. Push this repository to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "Add New Project"
4. Import your GitHub repository
5. Click "Deploy" - Vercel will automatically detect Next.js settings
6. Your site will be live in about 60 seconds!

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Pages

- `/` - Homepage
- `/programs` - Programs & Pricing
- `/team` - Meet the Team
- `/community` - Community & Testimonials
- `/blog` - Blog
- `/faq` - FAQ
- `/trial` - Free Trial Signup

## Customization

- **Logo**: Replace `public/logo.png`
- **Colors**: Edit `app/globals.css`
- **Content**: Edit files in `app/` and `components/`
